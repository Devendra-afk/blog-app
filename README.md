## BlogApp (MERN Stack)

BlogApp is a modern, full-featured blogging platform built using the **MERN stack** that allows users to create, read, update, and delete blog posts with a seamless writing and reading experience. It includes secure user authentication, role-based access control, a rich text editor, real-time comments, and dedicated dashboards for both users and admins.

##  Tech Stack

###  Frontend

* React.js
* Vite
* Tailwind CSS + Flowbite
* Redux Toolkit & Redux Persist
* Axios
* React Toastify
* Firebase OAuth

###  Backend

* Node.js
* Express.js
* MongoDB + Mongoose
* JWT Authentication + Bcrypt
* Multer + Cloudinary for image uploads
